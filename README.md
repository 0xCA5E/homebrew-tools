# 0xCA5E Tools

## How do I install these formulae and casks?

`brew install 0xca5e/tools/<formula>`

Or `brew tap 0xca5e/tools` and then `brew install <formula>`.

For casks:

`brew install --cask 0xca5e/tools/<cask>`

Or `brew tap 0xca5e/tools` and then `brew install --cask <cask>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "0xca5e/tools"
brew "<formula>"
cask "<cask>"
```
