# 0xCA5E Tools

## How do I install these formulae?

`brew install 0xca5e/tools/<formula>`

Or `brew tap 0xca5e/tools` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "0xca5e/tools"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
